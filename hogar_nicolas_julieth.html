💡 Principios del sistema

Estrategia Julieth (4h + 10-20 min/día)

Las 4h del sábado son el bloque fuerte. Los 10–20 min diarios son el anti-caos que mantiene el sistema funcionando entre semanas.

Estrategia Nicolás (tiempo flexible)

Las tareas de Nicolás están marcadas como Nicolás en el cronograma. Él las hace según disponibilidad, idealmente el mismo sábado.

Factor gato 🐱

Bandeja diaria (Nicolás). Aspirar cerámico 2×/semana por el pelo. Cepillar al gato 3×/semana reduce 60% del pelo acumulado.

Pisos mixtos + mampara de cristal

Baños/cocina = cerámica (mopa húmeda normal). Sala, comedor y habitaciones = laminado (mopa CASI SECA, producto específico, nunca agua abundante). Mampara: escurridor post-ducha siempre.

Orden correcto de limpieza — el sábado
1. Poner lavadora + aplicar antical baños
2. Ordenar + despejar + basuras
3. Baños en paralelo (20 min cada uno)
4. Cocina + zona de lavado
5. Sala, comedor y habitaciones
6. Aspirar/barrer toda la casa
7. Fregar: cerámica húmeda · laminado casi seco
8. Cierre: lavandería + gato + reset
Regla de pisos — lo más importante del apartamento

🧱 Cerámica — baños y cocina

✓ Mopa húmeda normal
✓ Agua abundante sin problema
✓ Desinfectante o limpiador cerámico
✓ Fregar 1× semana + barrer diario
✓ Juntas: limpieza mensual con ácido cítrico

🪵 Laminado — sala, comedor y habitaciones

⚠️ Mopa CASI SECA — escurrir muy bien
⚠️ Producto específico laminado (Bona o similar)
⚠️ Nunca agua abundante ni vapor
⚠️ Dirección de las lamas siempre
⚠️ Si cae agua: secar inmediatamente

Principios anti-acumulación

☀️ Mañana · Julieth (8–10 min)

● Airear habitación (ventana 5 min)
● Encimera cocina — paño húmedo rápido
● Lavar platos del desayuno
● Revisar zona de comida del gato

☀️ Mañana · Nicolás (5–6 min)

● Hacer la cama
● Vaciar bandeja del gato
● Cambiar agua del bebedero

🌙 Noche · Julieth (10–12 min)

● Limpiar encimera post-cena
● Fregar platos o lavavajillas
● Reset sala: cojines, mantas, mando
● Ropa sucia a la cesta

🌙 Noche · Nicolás (5–6 min)

● Escoba rápida en cocina (pelo de gato)
● Cerrar cajones y armarios abiertos
● Objetos fuera de lugar — a su sitio

Configuración Supabase

Conecta tu base de datos para guardar el registro de tareas de forma permanente.

🗄️ Paso 1: Crear proyecto en Supabase

1. Ve a supabase.com y crea una cuenta gratuita.

2. Crea un nuevo proyecto (el plan gratuito es suficiente).

3. Una vez creado, ve a Settings → API y copia:

  • Project URL (termina en .supabase.co)
  • anon public key

4. Ve al SQL Editor y ejecuta este script para crear las tablas:

-- Tabla de tareas completadas
CREATE TABLE IF NOT EXISTS task_log (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  task_id text NOT NULL,
  task_name text NOT NULL,
  who text NOT NULL,
  week_date date NOT NULL,
  completed boolean DEFAULT true,
  completed_at timestamptz DEFAULT now(),
  points integer DEFAULT 1
);

-- Tabla para shopping list (items marcados)
CREATE TABLE IF NOT EXISTS shopping_checked (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  item_id text UNIQUE NOT NULL,
  checked_at timestamptz DEFAULT now()
);

-- Tabla para rutinas diarias
CREATE TABLE IF NOT EXISTS daily_log (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  task_id text NOT NULL,
  who text NOT NULL,
  log_date date NOT NULL,
  completed boolean DEFAULT true,
  UNIQUE(task_id, log_date)
);

-- Política de acceso público (para demo)
ALTER TABLE task_log ENABLE ROW LEVEL SECURITY;
ALTER TABLE shopping_checked ENABLE ROW LEVEL SECURITY;
ALTER TABLE daily_log ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Public access task_log" ON task_log FOR ALL USING (true) WITH CHECK (true);
CREATE POLICY "Public access shopping" ON shopping_checked FOR ALL USING (true) WITH CHECK (true);
CREATE POLICY "Public access daily" ON daily_log FOR ALL USING (true) WITH CHECK (true);
